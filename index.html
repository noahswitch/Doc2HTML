Noah-Doc2HTML — Quick Reference

A live two-column editor. Left: rich document editing. Right: raw HTML source + live preview.

  • Formatting: use the toolbar, or keyboard shortcuts below.
  • Paste: paste Word / Excel HTML and it renders correctly. Right-click → Paste or Ctrl+V.
  • Copy HTML: copies the full HTML source to clipboard. Ctrl+Shift+C
  • Copy text: copies plain text content from the document.
  • Export: downloads a standalone index.html file. Ctrl+S
  • Change log: click the ⏱ icon to see a timestamped history of all edits.
  • Format: auto-indents and cleans up the HTML source.
  • Right-click in the editor for a context menu with all paste/copy options.

Keyboard shortcuts: Ctrl+B Bold, Ctrl+I Italic, Ctrl+U Underline, Ctrl+Z Undo, Ctrl+Y Redo
